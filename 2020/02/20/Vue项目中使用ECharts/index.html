接下来的内容为如何在你的Vue项目中添加ECharts插件

  1. 通过npm获取echarts
    npm install echarts –save

2.在你的Vue项目的入口文件main.js中引入echarts后修改原型链，可全局使用

  1. 在组件中为echarts创建一个具备宽高的DOM容器（！！！一点要有宽高，否则无法显示）
  2. 基于准备好的dom，初始化echarts实例

!!!初始化实例和绘制图表在vue生命周期mounted（）进行，保证当前页面元素都已经被加载完
