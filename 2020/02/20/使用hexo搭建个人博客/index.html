使用hexo搭建个人博客

搭建hexo个人博客并部署到github pages

  1. 需要用到的工具：git 、node.js
  2. node.js需到官网下载node.js （我已安装）
  3. hexo框架的安装需要借助npm包管理器，使用命令npm install -g hexo-cli 全局安装
  • 正式搭建hexo博客啦！！！
  1. 可以先mkdir创建一个文件夹，cd进入文件夹目录里，命令hexo init初始化一个博客。
  2. 启动博客：npm server
  3. 新建一篇博客文章：hexo n “文章标题”，文章文件在/source/_posts/目录下（可以直接打开文件写，也可以使用git bash用vim命令打开文章）
  4. 依次命令hexo clean清除缓存文件db.json和已生成的静态文件public、hexo g生成网站静态文件到默认设置的public文件夹、hexo server启动本地服务器预览
  • 将hexo部署到github pages！！！
  1. 首先得有github账号
  2. 在github上新建一个仓库create a new repository,重点！！！用户部署个人博客的github仓库的命名必须是 github昵称.github.io(比如我的昵称为ruanmanting，所以我的命名为ruanmanting.github.io)
  3. （在git bash下操作）在你的blog目录下安装一个插件npm install hexo-deployer-git –save
  4. 安装完后配置一个重要的参数，找到blog目录下的_config.yml,打开后配置一些参数：如图
  5. 修改完成后部署到远端hexo d
  6. 输入你的github账号和密码后开始往远端push
  7. push成功后输入你的网址即可啦！！！部署成功啦！！！
